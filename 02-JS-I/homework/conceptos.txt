- Variables
Es una especie de "contenedor" al que le ponemos un nombre y que nos sirve para guardar un valor. Este valor puede ser cualquier tipo de dato (números, letras, booleans, etc).

- Strings
Es un objeto para representar y manipular una cadena de caracteres.

- Funciones (argumentos, `return`)
La función es un procedimiento o un conjunto de instrucciones para obtener un resultado.
Los argumentos es una variable local que contiene el valor de esa función.
Return es la finalización de ejecución de la función y es la que da el resultado.

- Declaraciones `if`
Sirve para validar si una condición específica es verdadera. Si es falsa, da otro resultado.

- Valores booleanos (`true`, `false`)
Es para comparar sus dos valores true es verdadero y false es falso.
